# 贫困地区人口信息管理系统 java608

## 项目概述

贫困地区人口信息管理系统是一款基于前后端分离架构的系统，旨在实现对贫困地区人口信息的有效管理。系统采用了Vue.js、SpringBoot、MySQL等现代技术，具备完善的功能模块和灵活的权限控制。

## 技术栈

### 后端
- **框架**: SpringBoot
- **数据库**: MySQL
- **持久层**: MyBatis-Plus

### 前端
- **框架**: Vue.js
- **UI库**: ElementUI
- **HTTP库**: Axios
- **Node.js**: 用于前后端分离开发

## 功能模块

- **贫困人口管理**
- **精准扶贫管理**
- **贫困家庭支出统计**
- **特殊群体管理**
- **贫困户子女信息管理**
- **案件信息管理**
- **物资补助申领模块**
- **用户管理**
- **部门管理**
- **角色管理**
- **菜单管理**
- **日志管理**
- **数据字典管理**
- **文件管理**
- **图表展示**

## 系统角色

系统基于角色的访问控制，适用于管理员和工作人员，权限可精确到按钮级别。支持自定义角色和权限分配。

## 运行环境

- **后端**: JDK 1.8 或更高版本，SpringBoot 2.x
- **数据库**: MySQL 5.7 或更高版本
- **前端**: 现代浏览器（建议使用Chrome）
